# drishya-catalog

Public data catalog for the **Drishya** app — the channel and movie-home JSON the
app reads directly via `raw.githubusercontent.com`.

These files are generated and pushed automatically every 6 hours by a GitHub
Action in a separate (private) repository. Do not edit by hand — changes are
overwritten on the next refresh.

- `data/channels.json` — live TV channels
- `data/movies-home.json` — movie/show home feed
